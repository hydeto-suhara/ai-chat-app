# 🚀 須原専用AIチャットアプリ

スマホ・PCどちらからでもアクセス可能なAIチャットアシスタント

## 📱 使い方

### 初回設定
1. `index.html` をブラウザで開く
2. Gemini APIキーを入力（[Google AI Studio](https://aistudio.google.com/app/apikey)で取得）
3. 「保存して開始」をクリック

### 基本操作
- **メッセージ送信**: テキスト入力してEnterまたは送信ボタン
- **音声入力**: マイクボタンをクリック（Chrome/Edge推奨）
- **テーマ切替**: 月アイコンでダーク/ライト切替
- **会話クリア**: ゴミ箱アイコンで履歴削除

### スマホでの使用
1. スマホのブラウザでアプリにアクセス
2. ブラウザメニューから「ホーム画面に追加」
3. アプリのように使える（PWA対応）

## ✨ 機能

## 🚀 デプロイ方法

### ローカル実行
- `index.html` をブラウザで開くだけ

### GitHub Pages（推奨）
1. GitHubリポジトリを作成
2. コードをプッシュ
3. Settings → Pages → ブランチを選択 → Save
4. `https://[ユーザー名].github.io/[リポジトリ名]/` でアクセス可能
5. URLをスマホでブックマーク

## 🔐 セキュリティ注意事項

> **安全な設計**
> - APIキーはコード内に含まれていません
> - ユーザーが自分のAPIキーを入力する方式
> - APIキーはブラウザのlocalStorageにのみ保存されます
> - GitHubに公開しても安全です

> **注意点**
> - 絶対にAPIキーをコードにハードコードしないでください
> - 複数人で使用する場合は各自が自分のAPIキーを設定してください
> - localStorageはブラウザのデータ削除で消えるので注意
